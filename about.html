Why?

This webapp was created to fetch articles on the latest science news from Google News as a faster alternative to searching for it on Google. Even though the webapp scrapes content from Google News, I wanted the articles to be provided in an alternative manner. This webapp is intended to be a demonstration of my full stack development skills, being able to develop a frontend with the Bootstrap 4 framework, and backend with Express.js powered by Heroku.

How?

Frontend

The frontend was developed using HTML & CSS3 with the Bootstrap 4 framework alongside Javascript for functionality. When selecting a topic, the script sends a post request with an ID (required for searching Google News topics) to the backend to retrieve a list and displays it on the page. On load, the home page checks if the client is visiting the site for the first time and remembers what topic was selected last in the session to display articles of.

Backend

The backend was developed using the Node.js runtime environment with Heroku deploying the webapp. Express.js is used as the server framework with the Compression module serving all compressed responses. When sending a post request, the Request and Cheerio modules are used on the backend to scrape content from the provided Google News RSS feed URL, and send a list back to be displayed for the requested topic. To ensure a secure connection, a redirect from http to https takes place.
