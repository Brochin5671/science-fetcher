Science Fetcher

Fetch the latest science news from around the world!

Why?

I created this webapp to fetch articles on the latest science news as an alternative to searching for it on Google. This webapp is a demonstration of my programming skills, utilizing HTML and CSS for web-design, Javascript for scripting, and resources that aided in building the site.

How?

The webapp is hosted using Heroku, running on the nodejs environment with the Express module. HTML and CSS were used to design the webapp. Using Cheerio and the Request module in Javascript, the app web-scrapes articles off the Google News RSS feed and displays them on the site. This script is then bundled using Browserify in order to use the modules in-browser.
